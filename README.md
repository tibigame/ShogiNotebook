# Shogi Notebook

## Shogi Notebook とは

- まだbeta版です
- toolの枠を超えた1つのソフトウェア
- All Python3.6で書かれています
- 盤面をmatplotlibで表示することが可能
- usiエンジンを用いた検討、詰将棋解答など
- Jupyterを用いて将棋の定跡研究をノートブックにすることが可能
- usiエンジン同士の対局が可能
-- 合法手判定はしないのでやねうら王などのちゃんとしたエンジン同士を対戦させて棋譜収集を主目的としている
- sfenを用いた棋譜入出力
- 棋譜を分析して種々の統計情報を出力可能
- 棋譜の戦型判定が可能

## 構想など

- 振り飛車うんぬんの前に棋譜を解析できないとダメだろとコード書いてたら長大になった。
